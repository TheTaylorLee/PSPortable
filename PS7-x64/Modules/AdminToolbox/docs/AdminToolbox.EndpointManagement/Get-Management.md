# Get-Management

## SYNOPSIS

## SYNTAX

```
Get-Management [-computer] <Object> [<CommonParameters>]
```

## DESCRIPTION
Opens Computer management connected for another PC

## EXAMPLES

### EXAMPLE 1
```
Get-Management -computer PCName
```

Specify the computer you are connecting to.
