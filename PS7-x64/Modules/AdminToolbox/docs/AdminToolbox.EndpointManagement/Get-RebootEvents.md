# Get-RebootEvents

## SYNOPSIS

## SYNTAX

```
Get-RebootEvents [[-Server] <Object>] [<CommonParameters>]
```

## DESCRIPTION
Returns reboot history for specified Computer or Server

## EXAMPLES

### EXAMPLE 1
```
Get-RebootEvents -Server server.domain.tld
```
