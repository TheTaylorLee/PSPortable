# Get-FolderName

## SYNOPSIS

## SYNTAX

```
Get-FolderName [[-Description] <String>] [[-RootFolder] <String>]
```

## DESCRIPTION
Get a gui prompt for Specifying a folder name

## EXAMPLES

### EXAMPLE 1
```
$Folder = Get-Foldername
```

Creates a prompt to select a folder and then saves the path to a variable.
